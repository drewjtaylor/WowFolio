WeeklyDire

"WeeklyDire" is a fake news site (not a fake-news site.) Instead of doing countless small projects, I decided to spend time developing new features and learning new skills on a single project. As a result, what began as a small news/blog site with a few links turned into a full-stack cloud-deployed app. Developed with the MERN stack, it features lazy-loading, authentication using passport.js, a form for creators to add articles, a page for admin users to perform administrative functions, and Paypal integration allow users to upgrade to premium.

Formed Church

Formed Church was my first attempt at creating a website for a church in our area. The most exciting thing about this site isn't anything the user can see. This site is set up with CI/CD principles to allow for easy updates. Any time the main branch of github is updated, the code is automatically minified and uploaded to Netlify.com and published. Less time copying and pasting to deploy means more time doing and learning!
